# OrbitOps

A production-style multi-tenant SaaS dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Overview

OrbitOps is a modern SaaS admin platform concept focused on clean UI, scalable dashboard architecture, and real-world product patterns.  
It includes multiple dashboard pages, reusable components, responsive navigation, and interactive task management.

## Features

- Responsive dashboard layout
- Desktop and mobile sidebar navigation
- Overview page with analytics cards and project widgets
- Clients management page with table layout
- Projects page with progress tracking cards
- Tasks page with interactive filtering and completion toggles
- Reusable UI components and page structure
- Modern Neo Mint Tech design system

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- App Router

## Pages

- `/dashboard`
- `/dashboard/clients`
- `/dashboard/projects`
- `/dashboard/tasks`

## Design System

OrbitOps uses a custom **Neo Mint Tech** theme with:
- Mint primary tones
- Cyan accent colors
- Soft SaaS background palette
- Rounded cards and premium spacing
- Responsive app-shell layout
